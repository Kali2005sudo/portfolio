**Core Features**
- Learning Management System (LMS) with role-based flows and analytics
- IoT integration for sensor networks, automation, and edge workflows
- Weather Pro app showcasing clean UX, data visibility, and reliability

**Modern Web Optimization**
- WebP migration for portfolio imagery to cut payload size
- `image-set()` CSS with fallbacks for graceful cross-browser compatibility
- IntersectionObserver-driven scroll reveals for performant animations
- Lazy loading on non-critical images to speed initial paint

**Tech Stack**
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Python, PostgreSQL

**Installation**
1. `git clone <your-repo-url>`
2. `cd <your-repo-folder>`
3. `python3 -m http.server 5500`
4. Open `http://localhost:5500` in your browser
